Juptyer Notebook 的默认工作路径为 C:\Users\Username ，修改 Juptyer Notebook 的工作路径方法如下
（1）打开命令提示符，输入 jupyter notebook --generate-config
这个命令的作用是生成 Jupyter notebook 的配置文件。如果你是第一次运行，会直接生成这个文件。如果曾经运行过这个命令，就会覆盖原来的文件

（2）找到 jupyter_notebook_config.py 的路径并打此文件
