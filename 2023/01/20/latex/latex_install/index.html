# 一、Tex Live 与 Latex workshop 安装

Tex Live 可以从官网进行安装， Latex workshop 从 Vscode 的插件商城安装
⚠️⚠️⚠️需要注意的是， Latex workshop 插件需安装 8.8.0 版本，更高版本不稳定，会产生 bug

# 二、Tex Live 环境变量配置

在环境变量中添加 D:\texlive\2022\bin\win32 （需更改为自己的 texlive 安装路径）
