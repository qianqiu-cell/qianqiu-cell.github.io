# 一、生活篇

  1. 学会做饭：
    • 从理论上学习做饭的原理，并使用 latex 形成一篇理论文档
    • 掌握 5 道可以拿得出手的荤菜（要获得小小樊同志的认可）
    • 学会用刀切菜
    • 亲自下厨做一桌菜
    • 做一个点餐软件，每学会一道菜添加一个菜品（基于 pyqt5 ）
  2. 身体健康：
    • 年累计减重 5 公斤（目前 139 斤）
    • 年累计跑步 200 公里（在校每周 6 公里，要有跑步记录）
    • 年累计引体向上 1000 个（在校每周 30 个，要有 excel 表格记录）
    • 年累计俯卧撑 1500 个（每周 30 个，要有 excel 表格记录）
    • 慢慢调整作息至 12 点前睡觉
  3. 沈阳游玩
    • 去一次沈阳故宫、大帅府、中街
    • 该吃的都吃一边
    • 该聚的都聚一边
    • 跟着致远去旅游一次
  4. 终极目标
    • 赚 3w 人民币
    • 把房子装修成自己想要的风格
    • 成功搬进新房

# 二、学习篇

  1. 毕业设计
    • 争取尽早并高质量的完成毕业设计的核心算法（3 月中旬）
    • 做一个好看大气上档次的前端界面（基于 pyqt5 ）
    • 参与一项比赛并获得满意的奖项
    • 发表一篇符合自己水平的论文
  2. 研究生课程学习
    • 跟本科要求一样（自己满意即可，无需刻意刷分）
  3. 研究生科研竞赛
    • 在研究生入学之前掌握人工智能相关基础知识
    • 在研究生入学之前发表一篇人工智能相关论文
    • 在研究生入学之后发表另一篇论文（方向由老师提供）
    • 在研究生入学之前详细规划自己的三年研究生生活（包括科研规划，竞赛参与，工作方向）
  4. 社交能力
    • 研究生入学后参与竞选学生会及相关组织的成员、或班级委员
  5. 技能掌握
    • 能够熟练的使用 latex
    • 阅读能力提升，看到英语论文不发怵
    • 能够熟练的使用 python 进行人工智能方向开发
    • 闲暇时间学习 C++ ，对 C++ 有一个更好的掌握

# 三、其他

  1. 看更多视觉及剧情上爆炸的动漫（包括日本经典动漫以及国产动漫，希望国产动漫可以越来越好！！！）
  2. 在一定程度上减少对短视频的时间投入（短视频的质量亦有参差，要减少无意义的投入）
  3. 在校期间减少对于各类游戏的时间投入（真的很有助于学习质量的提升）
