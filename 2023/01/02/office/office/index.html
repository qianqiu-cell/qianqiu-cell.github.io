# 一、Word 使用技巧

# 1.1 刷新 word 中所有 mathtype 的公式编号

首先 CTRL+A 全选文档，接着按 F9 ，进行公式编号的刷新

# 1.2 一键调整 word 中 mathtype 公式的大小（以及格式刷后公式偏上问题）

（1）在 MathType 选项卡中选择格式化公式
（2）选择 MathType preference file ，点击 Browse...
（3）在 MathType 安装目录下进入 Preferences 文件夹，选择所需格式文件
（4）点击打开，点击 OK

# 二、Excel 使用技巧

# 2.1 使用 Excel 将两列内容合并为一列

然后按组合键 Ctrl+E ，在列中就自动合并了两列的内容，如下图所示

# 2.2 Excel 行列互换

直接对需要进行转置的区域进行复制，之后在空白区域进行粘贴，之后在粘贴的选项中选择转置选项，则粘贴的内容即为剪切板中内容的转置

# 2.3 Excel 加入斜线

（1）单调斜线分割成为两个区域
方法一：如下图方法一所示，依次输入 “编号人员”，编号选择下标，人员选择上标，最后在单元格格式中设置斜线；
方法二：如下图方法二所示，首先输入 “人员”，按下 ALT + ENTER 在单元格内换行，单元格文字选择左对齐，之后使用空格将第一行的 “人员” 两字编辑至合适位置，最后在单元格格式中设置斜线；
方法三：如下图方法三所示，使用四个单元格，在左下角输入 “编号”，在右上角输入 “人员”，对其他两个单元格设置斜线。

（2）双斜线分割成为三个区域
选择同一行的三个单元格，分别设置为左下、居中和右上，分别输入 “编号”、“任务” 和 “人员”，之后在插入选项卡中插入两条直线，并调整为如下图所示即完成
