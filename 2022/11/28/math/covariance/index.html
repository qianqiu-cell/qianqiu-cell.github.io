# 一、数学期望 E (x)

数学期望简称期望或均值，它是随机变量取值的平均值，数学期望的计算公式为
离散型：

E(X)=∑ixipiE\left ( X \right ) =\sum_{i}^{} x_{i}p_{i} E(X)=i∑​xi​pi​

连续型：

E(X)=∫−∞+∞xf(x)dxE\left ( X \right ) =\int_{-\infty }^{+\infty } xf\left ( x \right ) dx E(X)=∫−∞+∞​xf(x)dx

数学期望的性质

性质 1：设 c 是常数，则E(c)=cE(c)=cE(c)=c
性质 2：设 X 是随机变量，c 是常数，则E(cX)=cE(x)E(cX)=cE(x)E(cX)=cE(x)
性质 3：设 X，Y 是随机变量，则E(X+Y)=E(X)+E(Y)E(X+Y)=E(X)+E(Y)E(X+Y)=E(X)+E(Y)
性质 4：设 X，Y 是随机变量，若 X，Y 相互独立，则E(XY)=E(X)E(Y)E(XY)=E(X)E(Y)E(XY)=E(X)E(Y)

# 二、方差 D (x)

方差 D (X) 是 X 的取值相对其均值 E (X) 分散程度的度量值。D (X) 小，表示 X 的取值集中在 E (X) 附近；D (X) 大，表示 X 的取值相对于 E (X) 比较分散，方差的计算公式为

D(X)=E[(X−E(X))2]=E(X2)−(E(X))2D\left ( X \right ) =E\left [ \left ( X-E\left ( X \right ) \right ) ^{2} \right ] =E\left ( X^{2} \right ) -\left ( E\left ( X \right ) \right ) ^{2} D(X)=E[(X−E(X))2]=E(X2)−(E(X))2

方差的性质

性质 1：设 c 是常数，则D(c)=0D(c)=0D(c)=0
性质 2：设 X 是随机变量，c 是常数，则D(cX)=c2D(x)D(cX)=c^{2}D(x)D(cX)=c2D(x)
性质 3：设 X，Y 是随机变量，则

D(X+Y)=D(X)+D(Y)+2E[(X−E(X))(Y−E(Y))]D(X+Y)=D(X)+D(Y)+2E[(X-E(X))(Y-E(Y))] D(X+Y)=D(X)+D(Y)+2E[(X−E(X))(Y−E(Y))]

注：当 X 和 Y 相互独立时，D(X+Y)=D(X)+D(Y)D(X+Y)=D(X)+D(Y)D(X+Y)=D(X)+D(Y)

# 三、协方差 cov (X,Y) 和相关系数 ρ(X,Y)

# 协方差

协方差是反应随机变量 X,Y 之间关联程度的一个量，协方差的计算公式为

cov(X,Y)=E[(X−E(X))(Y−E(Y))]=E(XY)−E(X)E(Y)cov\left ( X,Y \right ) =E[(X-E(X))(Y-E(Y))]=E(XY)-E(X)E(Y) cov(X,Y)=E[(X−E(X))(Y−E(Y))]=E(XY)−E(X)E(Y)

如果令 X=Y，则有cov(X,X)=D(X)cov(X,X)=D(X)cov(X,X)=D(X)

协方差的性质

性质 1：cov(X,Y)=cov(Y,X)cov(X,Y)=cov(Y,X)cov(X,Y)=cov(Y,X)，特别的，cov(X,0)=0cov(X,0)=0cov(X,0)=0
性质 2：cov(aX,bY)=abcov(X,Y)cov(aX,bY)=abcov(X,Y)cov(aX,bY)=abcov(X,Y)
性质 3：cov(X+Y,Z)=cov(X,Z)+cov(Y,Z)cov(X+Y,Z)=cov(X,Z)+cov(Y,Z)cov(X+Y,Z)=cov(X,Z)+cov(Y,Z)
另外，设(X1,X2,⋯ ,Xn)(X_{1},X_{2},\cdots,X_{n})(X1​,X2​,⋯,Xn​) 是nnn 维随机变量，Xi X_{i}\spaceXi​  与Xj X_{j}\spaceXj​  的协方差都存在，cov(Xi,Xj)=cij,i,j=1,2,⋯ ,ncov(X_{i},X_{j})=c_{ij},i,j=1,2,\cdots,ncov(Xi​,Xj​)=cij​,i,j=1,2,⋯,n，则矩阵

C=[c11c12⋯c1nc21c22⋯c2n⋮⋮⋮cn1cn2⋯cnn]C=\begin{bmatrix} c_{11}& c_{12}& \cdots & c_{1n}\\ c_{21}& c_{22}& \cdots & c_{2n}\\ \vdots & \vdots & & \vdots \\ c_{n1}& c_{n2}& \cdots & c_{nn} \end{bmatrix}C=⎣⎢⎢⎢⎢⎡​c11​c21​⋮cn1​​c12​c22​⋮cn2​​⋯⋯⋯​c1n​c2n​⋮cnn​​⎦⎥⎥⎥⎥⎤​

# 相关系数

协方差 cov 的值一般受 X,Y 量纲的影响，通过把 X,Y 标准化，则可得到相关系数来消除这一影响，相关系数的定义为

ρ(X,Y)=cov(X∗,Y∗)=cov(X,Y)D(X)D(Y)\rho (X,Y)=cov(X^{*},Y^{*})=\frac{cov(X,Y)}{\sqrt{D(X)} \sqrt{D(Y)}} ρ(X,Y)=cov(X∗,Y∗)=D(X)​D(Y)​cov(X,Y)​

其中 X∗=X−E(X)D(X) X^{*}=\frac{X-E(X)}{\sqrt{D(X)} }\spaceX∗=D(X)​X−E(X)​ ，Y∗=Y−E(Y)D(Y) Y^{*}=\frac{Y-E(Y)}{\sqrt{D(Y)} }\spaceY∗=D(Y)​Y−E(Y)​ ，易知ρ(X,Y)\rho (X,Y)ρ(X,Y) 是无量纲的数

相关系数的性质

性质 1：ρ(X,Y)=ρ(Y,X)\rho (X,Y)=\rho (Y,X)ρ(X,Y)=ρ(Y,X)
性质 2：∣ρ(X,Y)∣≤1\left | \rho (X,Y) \right | \le 1∣ρ(X,Y)∣≤1
